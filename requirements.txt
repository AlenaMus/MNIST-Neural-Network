# GPU SETUP NOTES
# ============================================
# TensorFlow 2.10+ automatically detects and uses available GPUs
#
# GPU Requirements:
# - NVIDIA GPU with CUDA support
# - CUDA Toolkit 11.x or 12.x (compatible with TensorFlow version)
# - cuDNN library (compatible with CUDA version)
# - Minimum GPU Memory: 2 GB (recommended: 4+ GB)
#
# Verify GPU detection after installation:
#   python -c "import tensorflow as tf; print(tf.config.list_physical_devices('GPU'))"
#
# If no GPU detected:
# 1. Install NVIDIA GPU drivers: https://www.nvidia.com/Download/index.aspx
# 2. Install CUDA Toolkit: https://developer.nvidia.com/cuda-toolkit
# 3. Install cuDNN: https://developer.nvidia.com/cudnn
# 4. Ensure versions are compatible with TensorFlow
#
# For CPU-only installation (no GPU acceleration):
#   pip install tensorflow-cpu>=2.10.0
